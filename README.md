# taxi_bill

## Install 
```
start taxi_bill
```

## Requirement
* esx_billing https://github.com/ESX-Org/esx_billing

## Important
This script is written by myself and should not be sold for commercial use

## How to use it
```
/start  --start work
/stop   --stop work
/clear  --clear bill
/pay    --send bill
```
```
Config.Basic    --Basic charge
Config.KMprice  --Charge per kilometer
```
